## ⚙️ How to Use / Deploy
1. Clone the repo or upload via GitHub UI.
2. Enable GitHub Pages in **Settings > Pages**.
3. Choose source: `main` branch and `/root` folder.
4. (Optional) Create a `CNAME` file with your domain name (e.g., `accuflexpro.com`).
5. Update your DNS records to point to GitHub Pages IPs.
